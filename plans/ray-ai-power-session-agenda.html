The three tracks

Everything during the weekend should map to one of these. If an activity does not teach, improve life, or build confidence, cut it.

Track 1

Life upgrades

Build small systems that reduce friction: weekly planning, inbox processing, learning plans, reusable prompts.

Track 2

AI literacy

Learn context, memory, files, tools, verification, hallucination control, agent boundaries, and approval workflows.

Track 3

Build confidence

Ray leaves knowing how to ask better questions, critique output, run his agent, and keep improving after the visit.

Success looks like this

A clear finish line keeps the session from becoming random AI tinkering.

Prep checklist + starter prompt

Use this before Ray arrives so the weekend starts with momentum.

Night-before checklist

  • Ray’s Telegram bot responds.
  • OneDrive shared folder works for inbox/outbox files.
  • Ray brings 3 annoyances, 3 things he wants to learn, and 3 AI-help wishes.
  • Pick one main Day 2 build: default to AI inbox processor.
  • Agree on the safety pattern: draft first, review, then update.

Opening interview prompt

Interview me for 10 minutes and identify the top 5 opportunities where AI could improve my life. Focus on practical workflows, not generic productivity advice. Ask one question at a time. At the end, recommend the first 2 systems we should build this weekend.

Reusable workflow prompt

You are my practical AI workflow coach. Given this messy note, extract: 1) tasks, 2) projects, 3) decisions, 4) open questions, 5) suggested file updates. Draft first and ask for approval before changing my Life OS files.

Final handoff package

By the end, Ray should have a folder that makes the weekend continue after the weekend.

  • Use the AI inbox once per day for 7 days.
  • Do the 30-minute AI learning exercise daily.
  • Run a weekly review every Sunday.
  • Bring one real-life problem to the agent each week.
  • Keep a “what worked / what failed” note.
