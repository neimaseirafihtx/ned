Reality check: can Copilot actually do this?

Short answer: partly yes, but not as a guaranteed one-click pipeline. The weekend should prototype and validate each Microsoft tenant capability.

What looks feasible

  • Copilot in OneNote can help draft, create lists, organize information, and work from notebook context when the right Microsoft 365 Copilot experience is available.
  • Power Automate has a OneNote Business connector with triggers such as “when a new page is created in a section” and actions such as get page content, create page, update page content, and list sections/pages.
  • Power Automate Copilot can generate cloud-flow structure from natural language and suggest triggers/actions, then Ray still verifies connections and edits the flow.
  • Outlook / To Do / Planner actions are plausible through Microsoft connectors, assuming Ray’s tenant allows the required connections and data policies.

What must be tested

  • Copilot Pro vs Microsoft 365 Copilot: Ray’s “full pro Copilot” subscription may not equal enterprise Microsoft 365 Copilot inside his work tenant. Work-account licensing/admin policy is the gating item.
  • Handwriting extraction quality: OneNote/Copilot may handle typed notes better than messy handwritten ink. The first test must use Ray’s real iPad handwriting style.
  • Automation trigger delay: the OneNote connector is polling-based; Microsoft documents trigger polling frequency as 1 per 1200 seconds, so this may be a “process after meeting” workflow, not instant automation.
  • Security/DLP: Ray’s work tenant may block connectors, Outlook draft creation, Planner writes, or cross-app automation. No confidential notes should be copied into Hermes while testing.
Practical call: treat the OneNote system as a validated prototype target, not a promise. Build a manual-review workflow first; automate only after Ray proves OCR, connectors, and permissions work.

Evidence used for this reality check

  • Microsoft Learn describes Microsoft 365 Copilot as working across apps like Word, Excel, PowerPoint, Outlook, Teams, and OneNote, with OneNote experiences that can draft plans, generate ideas, create lists, and organize information. Microsoft notes app experiences may vary by licensing and tenant configuration.
  • Microsoft Learn’s OneNote Business connector lists triggers for new pages/sections and actions for creating pages, getting page content, getting pages/sections, and updating page content.
  • Microsoft Learn’s Power Automate Copilot documentation says cloud-flow Copilot can understand intent, create a flow based on a prompt, set up connections, apply parameters, and respond to requests to modify the flow.
